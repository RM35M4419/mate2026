# Matemática del Atelier - Práctica de Matemáticas

Este es una aplicación web para practicar matemáticas con la temática de **"Atelier of Witch Hat"** (Tongari Bōshi no Atorie). Todos los problemas y la interfaz están en español.

**En vivo en:** [https://RM35M4419.github.io/mate2026/](https://RM35M4419.github.io/mate2026/)

## Características

- **Temática Inmersiva:** La interfaz y los problemas están ambientados en el mundo de *Atelier of Witch Hat*.
- **Problemas Personalizados:** 
    - Los problemas narrativos usan personajes (Coco, Qifrey, Agott, etc.) y objetos del manga (pinceles de tinta mágica, sellos rúnicos, etc.).
    - Los problemas puramente matemáticos son presentados por los maestros del Atelier.
- **Flujo de Aprendizaje:** 
    - Un problema a la vez.
    - Opción de reintentar si la respuesta es incorrecta.
    - Opción de ver la respuesta correcta antes de pasar al siguiente.
- **Datos de Origen:** Utiliza los archivos en `/data` procesados para su uso en la web.

## Personajes y Elementos

- **Estudiantes:** Coco, Agott, Tetia, Richeh.
- **Maestros:** Qifrey, Olruggio.
- **Objetos:** Tinta mágica, Pinceles (Wand-pencils), Círculos mágicos, Sombreros puntiagudos.

## Estructura del Proyecto

- `index.html`: Estructura principal de la página.
- `style.css`: Estilos temáticos.
- `script.js`: Lógica de la aplicación.
- `data/problems.json`: Datos procesados.
